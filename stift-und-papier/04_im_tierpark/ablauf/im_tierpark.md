# Ablauf `Im Tierpark`

> **Ziele**: Bestehenden Code lesen und verstehen; Array-Methoden anwenden; Objekte besser verstehen

> **Themen**: Arrays, Loops, Objekte

1. Drucke im Vorhinein das Beiblatt [im_tierpark_arbeitsblatt.pdf](../material/im_tierpark_arbeitsblatt.pdf) halb so viel mal aus wie Personen vor Ort im Unterricht sind. 
2. Die Klasse soll sich in 2er-Gruppen aufteilen, den Code analysieren und die dazugehörigen Aufgaben lösen.
3. Sobald alle Gruppen fertig sind (max. 20') wird die Übung im Plenum besprochen.
