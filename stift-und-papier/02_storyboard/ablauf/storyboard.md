# Ablauf `Storyboard`

> **Ziele**: HTML & CSS analysieren; DOM und Event-Listener verstehen;

> **Themen**: DOM; Event-Listener; Zusammenhänge HTML, CSS & JavaScript

1. Drucke im Vorhinein das Beiblatt [storyboard_arbeitsblatt.pdf](../material/storyboard_arbeitsblatt.pdf) halb so viel mal aus wie Personen vor Ort im Unterricht sind. 
2. Die Klasse soll sich in 2er-Gruppen aufteilen. Es soll dann Schritt für Schritt in [script.js](../script.js) gearbeitet werden und jeweils skizziert werden, wie sich die Webseite verändert. Die auf dem Arbeitsblatt beschriebenen User-Interaktionen sollen auf der Skizze auch einfliessen. 
3. Sobald alle Gruppen fertig sind (max. 45') wird die Übung im Plenum besprochen.
